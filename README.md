# UnternehmenWebsite

Die Files für die Website der Firma

Notiz was zu machen ist:

- Vollständiges Impressum
- EU-DSGVO
- revDSG
- Urheberrechte für Bilder
- Markenrechte
- Unternehmenskommunikation

- favicon.ico, das kleine Icon im Browser Tab
- robots.txt, für die Suchmaschinen-Crawler
- sitemap.xml, für Übersicht aller wichtigen Seiten für Suchmaschinen
